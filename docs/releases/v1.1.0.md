# pTerminal v1.1.0 Release Notes

Release tag: `v1.1.0`  
Date: 2026-01-02

## Highlights

- Update flow now shows download/install progress with clear status messaging.
- Staged updates prompt for restart and automatically relaunch pTerminal to apply.
- Update checks only show the "Checking for updates..." message while a check is running.

## Update UX Details

- Check button hides once a release is available.
- Install flow shows byte progress during download and status during install.
- After staging, you are prompted to restart; selecting Restart relaunches the app.

## Validation

- `make fmt`
- `make vet`
- `go test ./...`

## Notes for Operators

- Updated binaries are staged as `pterminal.next` alongside the active binary.
- On restart, the staged binary replaces the current executable before launch.
