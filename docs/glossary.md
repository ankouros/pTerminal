# pTerminal - Glossary

Canonical terminology used across this repository.
If a term is ambiguous, clarify it here.

---

## H

### Host Key
The server's SSH identity fingerprint used to verify the host during connection.

---

## I

### IOshell
The local PTY driver used for specialized environments where native SSH is not desired.

---

## L

### LAN Sync
Encrypted peer-to-peer synchronization for team repositories on the local network.

---

## S

### SSH Agent
Local agent that holds private keys and authorizes SSH authentication without exposing key material.

---

## T

### Team Repository
The local directory where shared team configuration and files are stored and synced.

---

## W

### WebView
The embedded browser runtime used to render the UI (WebKitGTK on Linux).

---

## Glossary Governance

- New terms must be added here before use.
- Terms must be used consistently across documentation and code.
